# karute — 名医カルテAI / Doctor Memory AI

医師向けの診療支援ツール。カルテを取り込み、端末内で匿名化したうえで、症例プロファイル・鑑別診断候補・確認事項・診療文書ドラフトを提示する。

**本ツールは診療補助情報を提示するもので、医療機器認証を受けた診断装置ではない。確定診断・治療・処方の判断は医師が行う。**

## 構成

- `index.html` — フロントエンド一式（単一HTML、依存なし）
- `worker/` — Cloudflare Worker（Anthropic APIプロキシ。キーを隠し、CORS・モデル・max_tokensを正規化）
- `testdocs/` — 取込テスト用の架空カルテ・診断書6種（全て架空データ）

## 機能

- カルテ取込：テキスト貼り付け／画像・PDFアップロード
- 匿名化：送信前に端末内で氏名・生年月日（→年齢）・電話・住所・保険番号・患者ID・メールをマスク
- 症例プロファイル抽出（性別・年齢・疾患・受診理由・診断・処方・副作用リスク・想定治癒期間・注意事項）
- 鑑別診断候補（可能性・根拠・要確認点・確認/検査候補／確定は医師）
- 診療文書ドラフト生成（紹介状・患者説明文・診療録サマリ）
- JSON保存（医師確認メモ含む）

## 配信

1. Cloudflare Worker をデプロイし、`ANTHROPIC_API_KEY` を secret 登録
2. `index.html` の `API_ENDPOINT` を Worker の `/v1/messages` に差し替え
3. GitHub Pages もしくは Cloudflare で配信、独自ドメインを割当

症例DB（過去カルテの類似症例照合）は今後バックエンド（Worker + D1）で追加予定。
